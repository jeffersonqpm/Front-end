# Curso de Front-end — Treina Recife 🚀

## 📚 Sobre o curso

O curso de **Front-end do Treina Recife** é uma formação completa voltada para quem deseja iniciar ou se aprofundar no desenvolvimento de interfaces web modernas. Ao longo do programa, os alunos aprendem desde os fundamentos da web até a construção de aplicações interativas utilizando tecnologias amplamente utilizadas no mercado.

---

## 🧠 Tecnologias abordadas

### 🌐 HTML

* Estruturação de páginas web
* Semântica e boas práticas
* Acessibilidade básica

### 🎨 CSS

* Estilização de páginas
* Layouts responsivos (Flexbox e Grid)
* Animações e transições

### ⚡ JavaScript

* Lógica de programação
* Manipulação do DOM
* Eventos e interatividade
* Consumo de APIs

### ⚛️ React

* Criação de interfaces modernas
* Componentização
* Hooks (useState, useEffect, etc.)
* Gerenciamento de estado
* Integração com APIs

---

## 🛠️ Metodologia

O curso combina teoria e prática com:

* Aulas expositivas
* Exercícios práticos
* Projetos reais
* Desafios individuais e em grupo

---

## 🎯 Objetivos

Ao final do curso, os participantes serão capazes de:

* Criar páginas web responsivas e acessíveis
* Desenvolver aplicações interativas com JavaScript
* Construir interfaces modernas utilizando React
* Trabalhar com boas práticas de desenvolvimento front-end
